# Sitecore Cache tool
An improved version of Sitecore's `/sitecore/admin/cache.aspx` to better monitor usage and provide ability to clear individual caches.

The tool is written in inline .aspx so that it can be dropped directly on a running server without causing a restart.

The tool can be placed in any folder and is only accessible from localhost. Useful if all access to `/sitecore/admin` is blocked.
